# driver-monitoring
Driver monitoring by using the model from openpilot.

![](demo.mov)

## Installation

```
pip install -r requirements.txt
```

## Usage

**Stream from video**
```
./monitor.py -v videos/video_1.hevc
```

**Stream from camera**
```
./monitor.py -c
```
